Quản trị nội dung website

Khu vực này dùng để quản lý bài viết, hình ảnh và tin tức. Hệ quản trị khuyến nghị là Pages CMS, đăng nhập bằng GitHub và chỉnh trực tiếp nội dung trong repo.

Quản trị bài viết

Mở Pages CMS để đăng bài, sửa bài, tải ảnh lên thư mục public/uploads và quản lý nội dung trong src/content/posts.

Mở Pages CMS Mở repo GitHub

Đường dẫn quản trị

  • Bài viết: src/content/posts
  • Ảnh tải lên: public/uploads
  • Trang tin tức: /tin-tuc
  • Trang chi tiết bài viết: /tin-tuc/ten-bai-viet
